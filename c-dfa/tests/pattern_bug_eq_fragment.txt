# The pattern ls( =)[[safe::x]] produces a degenerate NFA with only 1 state
# while ls(z)[[safe::x]] works correctly with 12 states.
# 
# Root cause: The parser's position tracking after fragment parsing
# gets confused when the character before the fragment reference is '='
